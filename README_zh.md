# 算力前端网站说明文档

**其他语言: [English](README.md), [中文](README_zh.md).**

## 安装依赖

输入如下命令安装依赖:

```bash
pnpm install
```

## 构建开发服务器

输入如下命令启动开发环境:

```bash
pnpm dev
```

输入如下命令启动可选功能或添加新条目:

```bash
pnpm new
```

输入如下命令构建应用:

```bash
pnpm build
```

输入如下命令本地预览应用版本:

```bash
pnpm serve
```

有更多疑问，请查看 [Modern.js 说明](https://modernjs.dev/en).
